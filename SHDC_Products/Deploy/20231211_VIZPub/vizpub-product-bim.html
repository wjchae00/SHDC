기능 콘솔 INPUT

Export Image

  • CAD to Image 추출

    -mode IMG -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.jpg, *.png)" -img_format 0 -mode IMG -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.jpg, *.png)"

Convert VIZ

  • CAD to VIZ 변환

    -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.viz)" -export_viz_type 0 -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.viz)"

  • CAD to VIZM 변환

    -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizm)" -export_viz_type 1 -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizm)"

  • CAD to VIZW 변환

    -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizw)" -export_viz_type 2 -use_web_assembly t -export_vizw_for_vizwide3d t -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizw)" -use_web_assembly t -export_vizw_for_vizwide3d t

Export Structure

  • 형상 없이 모델 트리만 내보내기

    -mode STRUCT -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (VIZ, CAD)"

Export Metadata

  • CAD to Metadata 추출

    -mode META-i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.metadata)"

Export Attribute

  • CAD to 속성 TXT 추출

    -mode EXPATT -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.txt)"

Import Attribute

  • VIZ에 속성 TXT 가져오기

    -mode IMPATT -i "INPUT FILE PATH (*.viz)" -o "OUTPUT FILE PATH (*.viz)" -attribute_file_path "ATTR TEXT FILE PATH (*.txt)"

Remove Attribute

  • VIZ에서 속성 전체 삭제

    -mode DELATT -i "INPUT FILE PATH (*.viz)" -o "OUTPUT FILE PATH (*.viz)"

Edit Attribute

  • VIZ 내 속성 편집

    -mode EDITATTR -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -rule_file_path "C:\RULE.xml"

Model Rotate

  • VIZ 모델 회전

    -mode ROTATE -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -use_rotate_x 0 -use_rotate_y 0 -use_rotate_z 0

Model Color

  • VIZ 모델 색상 편집

    -mode COLOR -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -use_color_r 0 -use_color_g 0 -use_color_b 0 -use_color_a 0 -mode COLOR -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -color_xml_file_path "C:\COLOR.xml"

Export Model Boundbox Information

  • 모델 바운드 박스 정보 추출

    -mode BBINFO -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.txt)"

Export Boundbox Inner Model

  • 바운드 박스 내부 오브젝트 추출

    -mode INNER -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -use_boundbox_search_option 0 -use_boundbox_min_x 0 -use_boundbox_min_y 0 -use_boundbox_min_z 0 -use_boundbox_max_x 100 -use_boundbox_max_y 100 -use_boundbox_max_z 100

  • 모델 구조를 유지한 채, 바운드 박스 내부 오브젝트 추출

    -mode INNER -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -export_remain_structure_boundbox t -use_boundbox_min_x 0 -use_boundbox_min_y 0 -use_boundbox_min_z 0 -use_boundbox_max_x 100 -use_boundbox_max_y 100 -use_boundbox_max_z 100

  • 바운드 박스에 걸친 오브젝트 절단하여 추출

    -mode INNER -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -use_boundbox_cut_object t

Export Exterior

  • 모델 외곽 추출

    -mode EXPEXT -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)"

Export Simplified Model

  • 기본 간소화 모델 변환

    -mode SIMPLE -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)"

  • 숨기기 필터(*.txt)를 활용한 간소화 모델 변환

    -mode SIMPLE -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -hide_filter_object_file_path "C:\HIDE_FILTER.txt" -export_remain_structure_boundbox t -export_exclude_hidden_object_type 0

Merge Model

  • 모델 병합 (*.metadata)

    -mode MERGE -i "INPUT FILE PATH (*.metadata)" -o "OUTPUT FILE PATH (*.viz)"

Merge Node

  • 노드 병합 (*.xml)

    -mode MNODE -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -rule_file_path "C:\RULE.xml"

Export Node

  • 모든 노드 추출

    -mode NODE -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FOLDER PATH" -export_file_node_type 0

  • 파트 노드 추출

    -mode NODE -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FOLDER PATH" -export_file_node_type 1

  • 바디 노드 추출

    -mode NODE -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FOLDER PATH" -export_file_node_type 2

VIZPub.NET

VIZPub.NET은 Microsoft .NET Framework 4.0 기반의 Class Library 이며, 내부 코어 모듈은 C/C++/MFC 등의 언어로 개발이 되어 있습니다.

라이선스

VIZPub.NET 라이브러리는 SERVER 라이선스만 제공되며, 지정된 장비에서 무제한 실행 권한이 포함됩니다.

VIZPub

VIZPub은 3D CAD 모델을 SOFTHILLS Co., Ltd. (주)소프트힐스에서 제공하는 경량 파일 형식(*.VIZ)의 파일을 생성하고, 또한 생성된 경량 파일 형식을 활용하여 재활용 가능한 부가정보(혹은 Metadata)를 추출할 수 있는 변환기 소프트웨어 입니다.

사용 전 설치

VIZPub을 사용하기 위해서는 Microsoft Visual C++ 재배포 패키지를 설치해야 합니다.

설치가 필요한 버전은 아래와 같습니다.

  • Visual Studio 2010 (VC++ 10.0) SP1

  • Visual Studio 2012 (VC++ 11.0)

  • Visual Studio 2015 (VC++ 14.0)

최신 Windows Service Pack이 설치되지 않은 경우, 재배포 패키지 설치가 실패할 수 있습니다.

설치하는 운영체제의 플랫폼(Win32/x64)과 VIZPub 플랫폼에 맞는 재배포 패키지를 설치해야 합니다.

Visual C++ 재배포 패키지 최신버전 다운로드 사이트

라이선스

VIZPub 제품의 라이선스는 파일 형태로 제공되며, Node-Locked 종류만 지원하고 있습니다. (VIZPub.NET 제외)

라이선스 발행을 위해 사용하고자 하는 PC의 MAC ADDRESS 정보가 필요합니다.

[윈도우] -> [검색/실행] -> [명령프롬프트 / CMD.exe] -> "IPCONFIG /ALL" 을 입력

출력되는 정보 중 인터넷이 연결된 "이더넷 어댑터" 혹은 "무선 LAN 어댑터 Wi-Fi" 항목에서 "물리적 주소" 의 값

제공된 라이선스 파일은 VIZPub.exe 파일이 있는 동일 폴더 내에 제공된 VIZPub 제품 명과 동일한 파일 이름으로 존재해야 합니다.

EX) 제품명 : VIZPub BOM.exe

라이선스명 : VIZPub BOM.lic

VIZPub.Config

VIZPub.Config.exe는 VIZPub 변환기에서 사용하는 변환 옵션을 쉽게 파악하고 설정할 수 있도록 합니다. 설정된 옵션 파일은 “VIZPub.Config.ini”로 저장되며 VIZPub 상에서 해당 파일에 저장된 옵션들을 자동으로 적용하여 사용합니다.

VIZPub.Config.exe에서 설정 가능한 옵션 리스트는 콘솔 파라미터와 공유하고 있습니다. 이 곳에서 확인 가능합니다.

VIZPub.Config.ini는 VIZPub.exe와 동일한 경로 내에 존재해야 사용하실 수 있습니다.

Export XML

  • CAD to XML 추출

    -mode XML -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.xml)"

Export Image

  • CAD to Image 추출

    -mode IMG -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.jpg, *.png)" -img_format 0 -mode IMG -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.jpg, *.png)"

Convert VIZ

  • CAD to VIZ 변환

    -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.viz)" -export_viz_type 0 -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.viz)"

  • CAD to VIZM 변환

    -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizm)" -export_viz_type 1 -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizm)"

  • CAD to VIZW 변환

    -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizw)" -export_viz_type 2 -use_web_assembly t -export_vizw_for_vizwide3d t -mode VIZ -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.vizw)" -use_web_assembly t -export_vizw_for_vizwide3d t

Export Structure

  • 형상 없이 모델 트리만 내보내기

    -mode STRUCT -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (VIZ, CAD)"

Export Metadata

  • CAD to Metadata 추출

    -mode META-i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.metadata)"

Export Attribute

  • CAD to 속성 TXT 추출

    -mode EXPATT -i "INPUT FILE PATH" -o "OUTPUT FILE PATH (*.txt)"

Import Attribute

  • VIZ에 속성 TXT 가져오기

    -mode IMPATT -i "INPUT FILE PATH (*.viz)" -o "OUTPUT FILE PATH (*.viz)" -attribute_file_path "ATTR TEXT FILE PATH (*.txt)"

Remove Attribute

  • VIZ에서 속성 전체 삭제

    -mode DELATT -i "INPUT FILE PATH (*.viz)" -o "OUTPUT FILE PATH (*.viz)"

Edit Attribute

  • VIZ 내 속성 편집

    -mode EDITATTR -i "INPUT FILE PATH (*.viz, *.rvt, *.nwd)" -o "OUTPUT FILE PATH (*.viz)" -rule_file_path "C:\RULE.xml"

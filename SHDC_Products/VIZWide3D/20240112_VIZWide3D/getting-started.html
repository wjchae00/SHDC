HTTP 500.19 오류 해결 방법

IIS 관리자를 열고 VIZWide3D 실행 시 ".wasm" MIME 형식이 중복됐을 경우 500.19 에러 발생

VIZWide3D 폴더의 web.comfig 파일을 열고 ".wasm" mimeMap을 제외하여 해결
